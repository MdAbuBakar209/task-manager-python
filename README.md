# 📝 Task Manager CLI App

🚀 A simple and clean command-line Task Manager built using Python.

---

## 🎯 Features

- ➕ Add tasks  
- 📋 View all tasks  
- ⏳ View pending tasks  
- ✅ View completed tasks  
- ✔ Mark task as done  
- 🔄 Undo task (mark as pending)  
- ❌ Delete tasks  
- 🧹 Clear all tasks  
- 🚀 Mark all tasks as done  
- 💾 Persistent storage using JSON  

---

## 🛠 Tech Used

- Python  
- JSON (for data storage)  
- Colorama (for colored UI)

---

## 💡 What I Learned

- Python fundamentals  
- File handling  
- CLI-based application design  
- Improving user experience (UI in terminal)
